# 🧩 MBM – Multiple Business Management System

## 📌 Project Overview
MBM (Multiple Business Management System) is a full-stack web application designed to manage multiple businesses from a single platform. It helps in handling employees, projects, transactions, and business analytics efficiently through a centralized dashboard.

The system is built using **React (Frontend)** and **Spring Boot (Backend)** with **MySQL** as the database.

---

## 🚀 Features

### 👨‍💼 Business Management
- Create and manage multiple businesses
- Centralized business tracking system

### 👥 Employee Management
- Add, update, delete employees
- Assign employees to businesses

### 📁 Project Management
- Create and manage projects
- Assign projects to employees/businesses

### 💰 Transaction Management
- Track income and expenses
- Maintain financial records

### 📊 Dashboard
- Business overview
- Revenue analytics
- Profit & expense tracking
